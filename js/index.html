Primrose v1.1.x Follow Primrose on Github Follow Primrose on NPM

A source code text editor that renders in an HTML5 canvas. Useful for texturing objects in WebGL apps.

Features

  • International keyboard support (left-to-right rendering only)
  • Wide Unicode character-aware
  • Line numbering
  • Color theming
  • Syntax highlighting for:
    • JavaScript,
    • HTML, and
    • BASIC
